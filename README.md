### 项目介绍

`spring boot example` 是一个用来学习并实战 `spring boot` 的项目，目前总共包含 **`1`** 个集成demo，已经完成 **`1`** 个。
该项目已成功集成：

### 开发环境

- **JDK 1.8 +**
- **Maven 3.6 +**
- **IntelliJ IDEA ULTIMATE 2022.1.2 +**
- **Mysql 5.7 +** (8.0以上的版本也行，但是可能出现一些兼容性问题)
- **spring boot version 2.7.4**
